Jarvis - Systems Administration ChatOps

A bot built into Slack that allows users to gather information from Virtual Machines. Such information includes computer status, on/off/restart, network information, printer information, software installation via chatbot, and much more. Project uses Slack RTM API, Slack's Easy-Peasy-Bot API as well as Javascript and Bash scripting for cronjobs.

Simon Says - Embedded Systems Project

Using the ATmega1284 processor, I implemented a Simon Says game. I used an 8x8 LED Matrix, 4 large SparkFun buttons for easy process, LED Screen with custom characters, EEPROM for saving data, as well as timer/period manipulation. All code was built in AVR Studio using built in libraries such as air/io.h, air/interrupt.h, stdio.h, avr/eeprom.h. All of the components were built in a breadboard.

GitHub Page YouTube Demo

RSHELL

RSHELL is my own built unix shell. Using version control from GitHub, vim, and debugging tool GDB, RSHELL was built to use commands such as 'ls', 'cd', 'make', 'touch', run scripts, I/O redirect, and minor implementations of piping.
